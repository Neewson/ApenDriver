# ApenDriver - Plataforma de Armazenamento em Nuvem

ApenDriver é uma plataforma de armazenamento em nuvem para gerenciar seus arquivos de mídia, incluindo áudios, vídeos e imagens.

## Funcionalidades

- **Gerenciamento de Arquivos**: Upload, visualização e download de arquivos de mídia
- **Player de Áudio**: Reprodução de arquivos de áudio com controles completos
- **Player de Vídeo**: Reprodução de arquivos de vídeo com controles completos
- **Galeria de Imagens**: Visualização de imagens com modo de apresentação
- **Dashboard**: Visão geral do uso de armazenamento e atividade recente
- **Interface Responsiva**: Funciona em dispositivos desktop e móveis

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome
- Chart.js

## Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador
3. Faça login com qualquer email e senha (este é um protótipo sem backend)
4. Explore as diferentes seções da plataforma
